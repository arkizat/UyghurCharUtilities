## 1. ئاساسىي رايوننى كېڭەيتىلگەن رايون كودىغا ئالماشتۇرۇش ئۇسۇلى(getUyPFStr)

ئۇيغۇرچە ئالماشتۇرۇش تۈر(类، Class) ئىسمى:UyghurCharUtilities

```php

//$text Almashturmakchi Bolgan Text

$uyghur->getUyPFStr($text);
```
